# Backtest / research reports

Drop Strategy Tester exports and analysis notes here.

## Naming

```text
REPORT-bt-YYYY-MM-DD-XAUUSD-M5-<variant>.md
```

Examples:

- `REPORT-bt-2026-07-15-XAUUSD-M5-E-Fixed-2R.md`
- `REPORT-bt-2026-07-15-XAUUSD-M5-E-Trail.md`
- `REPORT-wf-2026-07-20-walkforward.md`

## Minimum fields per report

| Field | Required |
| --- | --- |
| Period (IS / OOS) | yes |
| Variant (params snapshot) | yes |
| Trades, PF, Net, Max DD | yes |
| Winrate, avg R | yes |
| Notes / red flags | yes |
| Link raw HTML/XML if any | optional |
